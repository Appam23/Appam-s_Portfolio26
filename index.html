About Me

Hi, I'm Leisan I code in three languages: JavaScript, fabric, and sound. I know that sounds unusual for a web developer, but here's the thing: creativity doesn't live in silos. I'm a Web Developer studying at Brigham Young University-Idaho, where I've built a strong foundation in HTML5, CSS3, JavaScript, React, Vue, and Angular. But my journey with design started long before I wrote my first line of code. The Fashion Connection When I'm draping fabric and constructing garments—like the custom denim ensemble I showcased at our Winter Fashion Show—I'm thinking about proportion, color harmony, texture, and how the wearer will feel. Sound familiar? Those are the same considerations I bring to web design: layout, color theory, user experience, and emotional engagement. Fashion taught me that details matter. A seam that's off by a millimeter can ruin the entire garment. A button placed wrong disrupts the visual flow. The same is true in web development—a poorly timed animation, an inaccessible color contrast, or a clunky navigation can break the user experience. The Music Connection When I produce music, I'm layering sounds, creating rhythm, building tension and release. I'm thinking about the journey I want to take the listener on. Web development is no different. Every website is a journey. Every click, scroll, and interaction is part of a composition that should feel natural, intuitive, and—when done right—delightful. Where It All Comes Together This multidisciplinary background means I don't just build websites—I craft experiences. I think about: How color palettes evoke emotion (like choosing fabric) How users flow through a site (like structuring a song) How micro-interactions create delight (like finding the perfect button placement on a coat)
